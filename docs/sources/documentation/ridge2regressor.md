### Ridge2Regressor


```python
ahead.Ridge2.Ridge2Regressor.Ridge2Regressor(
    h=5,
    level=95,
    lags=1,
    nb_hidden=5,
    nodes_sim="sobol",
    activation="relu",
    a=0.01,
    lambda_1=0.1,
    lambda_2=0.1,
    type_pi="gaussian",
    B=100,
    date_formatting="original",
    seed=123,
)
```
